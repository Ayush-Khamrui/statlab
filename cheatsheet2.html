2 Mathematical Expectations

Expected
Value $\mu$
Discrete $$E(X) = \sum_{x} x \cdot P(x)$$
Continuous $$E(X) = \int_{-\infty}^{\infty} x \cdot f(x) dx$$
Function $E[g(X)]$: $\sum g(x)P(x)$ | $\int g(x)f(x)dx$
Variance
$Var(X)$
Definition: $Var(X) = E[(X-\mu)^2]$ Shortcut: $Var(X) = E(X^2) - [E(X)]^2$
Discrete $\sum_{x} (x-\mu)^2 P(x)$
Continuous $\int_{-\infty}^{\infty} (x-\mu)^2 f(x) dx$

Linearity Properties (Crucial!)

Expectation
$E(c) = c$
$E(aX) = aE(X)$
$E(aX \pm b) = aE(X) \pm b$
$E(X+Y) = E(X) + E(Y)$
Variance
$Var(c) = 0$
$Var(aX) = a^2Var(X)$
$Var(aX \pm b) = a^2Var(X)$
(if X, Y independent)
$Var(X \pm Y) = Var(X) \mathbf{+} Var(Y)$

3 Naive Bayes Theorem

The "Naive" Assumption

Assumes all input features $(x_1, x_2 ... x_n)$ are Conditionally Independent of each other given the class label $y$.

$$P(X_1, X_2 | y) \approx P(X_1|y) \times P(X_2|y)$$

Classification Rule (MAP)

Maximum A Posteriori

$$\hat{y} = \argmax_{y} P(y) \prod_{i=1}^{n} P(x_i | y)$$
The Naive Bayes Master Formula
$$P(y | x_1, x_2, \dots, x_n) = \frac{P(y) \prod_{i=1}^{n} P(x_i | y)}{P(x_1, x_2, \dots, x_n)}$$
Posterior
Prior $\times$ Likelihoods
Evidence (Constant)
Feature $x_1$
Feature $x_2$
Feature $x_3$
Class $Y$

Note: In practice, denominator $P(x)$ is ignored for classification because it's the same for all classes.

4 Laplace (Additive) Smoothing

⚠️

The Zero-Frequency Problem

If a specific feature value never occurs with a specific class in the training data, its conditional probability becomes $0$.

Because Naive Bayes multiplies all likelihoods together, a single zero wipes out the entire probability equation.

$0.4 \times 0.7 \times \mathbf{0} \times 0.9 = \mathbf{0}$

The Solution: Laplace Smoothing

We add a small constant $\alpha$ (usually $1$) to every count, so no probability is ever strictly zero. To keep probabilities summing to 1, we add $\alpha \times d$ to the denominator.

$$P(x_i | y) = \frac{\text{Count}(x_i, y) + \alpha}{\text{Count}(y) + \alpha \cdot d}$$
$\alpha$ : Smoothing parameter (Laplace = 1)
$d$ : Number of possible distinct values for feature $x_i$
📝
Text Classification Context: In NLP (like Spam Detection), $d$ becomes $|V|$ (the size of the entire Vocabulary). The formula becomes: $P(w_i|C) = \frac{\text{count}(w_i, C) + 1}{\text{count}(C) + |V|}$
🚨

🚨 Advanced Exam Traps

1. Variance of a constant is ZERO.
$Var(5) = 0$
2. $Var(aX) = a^2 Var(X)$. Don't forget to square the constant when pulling it out!
3. Variance always adds.
$Var(X - Y) = Var(X) \mathbf{+} Var(Y)$
4. Forgetting the Prior in Bayes. You must multiply the likelihoods by $P(Class)$ to get the numerator!
5. Laplace Denominator: Add $|V|$ (total distinct words in ALL classes), not just words in that class.
6. Underflow Error: Multiplying many small probabilities equals 0 in computers. Use Sum of Logs instead.
